## Getting Started

### Prerequisites

- Node.js 18 or later
- PostgreSQL database
- OpenAI API key

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Set up environment variables in `.env.local`
4. Start the development server:
   ```bash
   pnpm dev
   ```
